#  Flappy Bird Clone (Python)

A simple Flappy Bird clone built using Python.
This project focuses on core game logic, event handling, and basic physics simulation.

##  Features

* Smooth bird movement with gravity mechanics
* Obstacle (pipes) generation and collision detection
* Score tracking system
* Continuous game loop with real-time updates

##  Tech Stack

* Python
* Pygame

##  What I Learned

* Handling game loops and real-time rendering
* Implementing collision detection
* Managing user input and physics-based movement
* Structuring small-scale projects cleanly

##  How to Run

```bash
pip install pygame
python game.py
```
